# Product Overview

## ToPoint Notebook

A privacy-focused desktop application for managing text and markdown notes with strong encryption guarantees.

### Core Value Proposition
- **Self-custody data management**: All notes stored in a single encrypted file under user control
- **Privacy-first design**: No external services, no network transmission of user data
- **Simple yet secure**: Easy-to-use interface with industry-standard encryption (AES-256)

### Key Features
- Create, edit, and organize text and markdown notes
- Import/export individual files while maintaining format integrity
- Full-text search across all notes
- Markdown syntax highlighting and preview mode
- Automatic backups with integrity verification
- Auto-lock functionality for enhanced security

### Target Users
Users who prioritize data privacy and want complete control over their personal notes and documents without relying on cloud services.

### Security Model
- Single encrypted file contains all data and metadata
- Password-based authentication with secure key derivation
- No plaintext storage of sensitive information
- Automatic memory clearing when locked
- Data integrity verification on every access

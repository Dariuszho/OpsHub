# XML ↔ JSON Converter (Standalone HTML)

Zero-dependency, single-file web application. No installation required.

## Usage

Simply open `index.html` in any modern web browser.

### Windows
Double-click `index.html` or right-click → Open with → Browser

### Linux
```bash
xdg-open index.html
# or
firefox index.html
# or
google-chrome index.html
```

## Features

- Bidirectional conversion: XML ↔ JSON
- Upload files from local machine
- Download converted results
- Clear content with one click
- Works completely offline
- No server or installation needed

## Limitations

- URL fetch only works with CORS-enabled endpoints
- For fetching from arbitrary URLs, use the Python or React versions which include a backend proxy

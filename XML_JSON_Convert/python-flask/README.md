# XML ↔ JSON Converter (Python Flask)

Server-based web application using Python and Flask.

## Requirements

- Python 3.7+

## Installation

```bash
pip install -r requirements.txt
```

## Usage

### Windows
```
run.bat
```

### Linux
```bash
chmod +x run.sh
./run.sh
```

Open http://localhost:5000 in your browser.

## Dependencies

- Flask - Web framework
- xmltodict - XML/JSON conversion
- requests - HTTP requests
- waitress - Production WSGI server
